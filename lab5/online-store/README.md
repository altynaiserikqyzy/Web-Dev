# Online Store - Lab5

Angular project simulating an online store with categories and products.

## Project Structure
## How to Run

git clone https://github.com/altynaiserikqyzy/Web-Dev.git
cd lab5/online-store
npm install
ng serve


Features
	- Display products by category
    - Image gallery with main image and thumbnails
    - Like button to increment product likes
    - Delete button to remove products from the list
    - Share buttons for WhatsApp and Telegram
    - Responsive product cards with hover effects
    - Empty category message: “No products in this category”
Notes
	•	Do not push the node_modules/ folder. It is already included in .gitignore.
	•	All products are linked to real items from Kaspi.kz.
	•	This project uses Angular standalone components (standalone: true) for ProductList and ProductItem.
